# Rahul Gupta - Developer Portfolio

> 🍱 A premium, highly-responsive Bento-grid developer portfolio featuring real-time competitive programming stats, dynamic theme switching, and a chunky retro-modern aesthetic. Built with Vanilla HTML/CSS/JS for maximum performance.

## 🌟 Features

- **Bento Grid Architecture**: A perfectly aligned grid system that flawlessly adapts from mobile devices to ultrawide screens.
- **Theme Engine**: System-aware Dark/Light mode toggle with custom HSL/OKLCH color palettes and persistent UI states.
- **Competitive Programming Stats**: Dedicated section highlighting problem-solving achievements across LeetCode, Codeforces, and CodeChef.
- **Scrollable Sections**: Efficient content containment with vertically scrollable project arrays and experience timelines.
- **Chunky Aesthetic**: Premium UX featuring custom micro-animations, chunky drop shadows, and high-contrast borders.
- **Zero Dependencies**: Lightning-fast load times utilizing pure HTML5, CSS3, and Vanilla JavaScript.

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3, Vanilla JavaScript
- **Typography & Assets**: Geist & Geist Mono Fonts, Remix Icons
- **Design Patterns**: Bento UI Layouts, CSS Grid/Flexbox, CSS Variables for dynamic styling

## 🚀 Getting Started

To view or modify this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rgcodexd/portfolio.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd portfolio
   ```

3. **Run the project**:
   Simply open `src/index.html` in your web browser, or use a local development server (like VS Code's Live Server) for hot-reloading:
   ```bash
   # Using Python (optional)
   cd src
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
